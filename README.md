# 数字小猫产品分析专家 (Digital Kitty Analyst)

一个为 **AI 数字宠物 / 虚拟陪伴类产品**早期阶段（构思 / 市场验证 / 寻找 PMF）打造的 Claude Skill。基于 25 个经典心理学与思维模型，针对数字小猫产品场景做了深度定制。

## 这个 skill 解决什么

如果你正在构思、调研、或刚启动一款数字小猫 / AI 陪伴 / 虚拟养成类产品，这个 skill 帮你：

- **需求验证**：用 JTBD + 第一性原理拆解"用户为什么真的需要数字小猫"
- **用户洞察**：理解情感产品的留存机制（拥有权效应、宜家效应、峰终法则等）
- **竞品分析**：用幸存者偏差、林迪效应找差异化定位
- **商业模式**：用锚定、心理账户、好-更好-最佳设计可成立的定价
- **增长留存**：用转换成本、复利效应、飞轮效应预演长期增长

## 设计立场

这个 skill 不是通用营销心理学知识库。它有几个明确的偏见，先告诉你：

1. **数字小猫是情感产品，不是 AI 产品** — 用户买的是被陪伴的感觉，不是技术
2. **失败的最大原因不是 AI 不够好，是新鲜感过后没东西** — 留存设计 > 功能堆叠
3. **早期不要相信数据，要相信和真实用户的对话** — DAU 200 可能比 DAU 20 更没价值
4. **不做"全能 AI 宠物"** — 选一个细分定位深耕

如果你不认同这些立场，这个 skill 可能不适合你。
